速刷红魂

通关一周目、进终业 DLC 后,才有真正的大批量刷法:红魂挑战叠满 + 刷序章,一把约 300 红魂、13 分钟。

佛堂里花红魂解锁卖红魂挑战的 NPC(建议先把科技树点满再解锁)。

贰

出门前疯狂叠挑战

每"选择"一个挑战就 叠一层 debuff + 一份红魂奖励。界面只有 5 个展示位,但选完立刻刷新顶上,所以可以连选 n 个不断叠加,叠到你的流派还能轻松清的数量(十几个),奖励就堆到三百多。

叁

专挑序章刷

序章最短,叠满奖励却只用打一小段,红魂/分钟最高。用强力流派一路清掉,结算领魂,约 13 分钟一把,重复。

这些挑战拖节奏/致命,跳过:以瞬代步(基本没法走路)、命定死亡(不熟 BOSS 别碰)、索命鸳鸯(成型前比 BOSS 还狠)、灵魂重担、混沌现实。位子卡到难的就卡死重置刷掉。
早期没 DLC 时的过渡刷法:打过第一章后,利用"离开场景前暂停 → 重新开始"反复刷第一章,每把拿 5–7 红魂;红魂是永久的,退出/重开不会丢。遇到打不过的强敌选"逃跑/放弃",还会按剩余血量百分比返还红魂。

天赋加点

本体树和终业树是两套独立盘,红魂越多开局加成越强。这两套的加点顺序如下。

本体树 · 红黄绿

新手地基

① 绿色系前半页 — 最优先。开局没减伤、血上限仅 100、不能精炼加血,容易被走火反伤打死;前半页补容错,还解锁神器 玄武之魂。
② 黄色系点满 — 出门自带两圣物 + 一本秘籍,开荒最大一次提升。
③ 红色系 — 攻击/伤害向,拉面板。
④ 绿色后半页 + 杂项 收尾。

树不大,速刷细雪几盘就全满,顺序只影响最初几十红魂。

终业树 · 命杀炼

DLC 进阶

命=防御(生命/减伤/葫芦次数)、杀=输出、炼=运气/死后货币存留等杂项。
顺序:优先杀 → 其次命 → 炼最后(炼前四排前期没用,别浪费)。
杀天赋:左列中列=近战,右列=远程;飞剑远程通关效率低,建议走近战。

魂多势众是发育型,平民不太建议学,容易浪费一个秘籍位。

伍

核心机制词典

看懂这些,你就懂了这游戏的伤害逻辑。

走火入魔

进攻

+300% 全攻击,-300% 防御(实为大幅增加受到的伤害),击杀回复。伤害核武,但裸点会暴毙。

配玄武 / 绿天赋容错 / 空明无敌一起用才安全。

玄武之魂

攻防一体

把防御转成伤害——每 +1% 防御 → +1.5% 伤害。既扛揍又增伤,无影、日月等流派的地基。

绿色系前半页解锁;和走火、猪心不冲突,可叠乘。

憎恨之心

无副作用

核心/功效位 +100% 全攻击,没有任何负作用。通用、安全、强,开荒最值得 SL 的核心。

需先解锁;没有就退求三色结晶 / 狼王白鬃。

母神旧印

无限R

提供 无限 R(收剑/大招) 的关键圣物,多个 DLC 流派的发动机。

刷完书别急着出门,先去圣物箱定向锁它,否则没有无限 R。

踏风草

减CD

冲刺/瞬身能削减收剑 CD,配合后变“无限滑铲”。这就是为什么火2、雷1 收剑 CD 长却不顿。

CD 嫌长别干等:喷酒+平A补刀+狂冲刺,踏风草帮你把 CD 滑没。

真伤 / 百分比

打BOSS

蛇火百分比扣血(吃 BOSS 大血条)、龙火/真火无视防御真伤。这正是酒系打 BOSS 比纯机制流舒服的原因。

打 BOSS 走蛇酒(配雄黄)伤害最高,龙酒无视防御更稳。

堕龙

近战必带

+100% 飞剑攻击 + 召唤圣剑助战,近战流派的出门首选。

常规掉率为 0,只能在补给站用红魂随机刷新获得。

精炼方向

选攻击

所有非单次加成都是累乘,10%+10% > 20%。所以精炼一律加攻击,别加生命。

精炼放最后做——SL 退出重进会让精炼的攻击砍半、生命清零。
陆

实用技巧 · SL

回报最高的几个操作。前两条几乎每把都用得上。

SL 刷随机产物

开局圣物、技能书、中途秘籍/骷髅房、大女鬼商店、品质升级,不满意就右上角三条杠 → 重新开始反复刷。品质升级实时随机,重开可确保成功。

SL 掉属性陷阱

每次离开游戏再进,上次精炼的生命消失、攻击砍半。所以精炼要放在确定不再 SL 之后做,开荒长局尤其注意。

重铸武器换轨

直接重铸出 A1-A4 都不要,就离开重进、先 3 选 1 拿圣物,再重铸会变成 B1-B4 新序列,想抠特定武器(如太祖)很有用。

刷流派专属秘籍

章节最终 BOSS 掉的秘籍奖池更小、专属占多数。想玩某流派(如蛇酒)就反复打序章 BOSS,拿到再进第一章;房间怪物随机,刮不动可退出重进。

柒

见闻录 & 封印

见闻录补全剧情 + 解锁永久加成,本体图鉴在床左边。一部分藏在地图角落,其余靠击杀概率掉落。

新手大坑 — 这几个别手贱解锁:见闻录能解锁圣物【血玉髓】【星辰之玉】和武器【咒杀】,但它们十分鸡肋,解锁后只会污染奖池,让你想要的好东西更难刷出来。不是为了刷成就就别点。
封印 = 提纯奖池。圣物/武器封印后在局内不再掉落,能大幅提高刷到目标的概率;但前期红魂少时先别封印。
